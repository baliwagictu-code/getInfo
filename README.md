# 🖥️ System Information Collector (PowerShell)

This project is a PowerShell script that collects detailed computer information and sends it to a Google Sheet using a web API.

## 📌 Features

- User input:
  - Full Name
  - Property Number
- Device type selection:
  - Laptop
  - Desktop
- Collects system details:
  - Computer Name
  - Motherboard (brand + model + UUID)
  - Device Serial Number
  - CPU
  - RAM (per stick with brand and serial)
  - Storage (type, size, serial)
  - GPU
  - Operating System
- Automatically sends data to Google Sheets

---

## ⚙️ Requirements

- Windows OS
- PowerShell (default in Windows)
- Internet connection (for sending data)
